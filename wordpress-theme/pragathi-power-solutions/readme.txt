A custom solar-energy business theme for Pragathi Power Solutions, with brand
colours sampled directly from the PPS logo.

== Description ==

A self-contained, page-builder-free WordPress theme that reproduces the Pragathi
Power Solutions marketing site:

* Home, About, Services, Projects, Gallery and Contact pages — fully designed.
* Brand palette taken straight from the logo: navy #1d2f7e, sun amber #ffa212,
  tagline red #df1c00, panel cyan #29abe2, violet #6a2c91.
* Interactive savings calculator, FAQ accordion and a contact form that hands
  the enquiry to WhatsApp.
* Editable company details (phone, email, address, social links, logo) via
  Appearance > Customize > "Pragathi: Company Details".
* No external build step, no jQuery, no plugins required.

== Installation ==

1. In WordPress admin, go to Appearance > Themes > Add New > Upload Theme.
2. Choose pragathi-power-solutions.zip and click Install Now, then Activate.
3. On activation the theme automatically:
   - creates the Home, About, Services, Projects, Gallery and Contact pages,
   - sets Home as the static front page,
   - builds a "Primary" menu and assigns it to the header.
4. Visit the site — everything is live. Edit contact details under
   Appearance > Customize > "Pragathi: Company Details".
